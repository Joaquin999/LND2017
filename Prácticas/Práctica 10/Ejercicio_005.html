La instrucción a:link indica que todos los elementos <a> con un enlace externo cumplirán las instrucciones

La instrucción a:visited indica que todos los elementos <a> con un enlace externo que ya haya sido visitado cumplirán las instrucciones

La instrucción a:hover indica que cada vez que todos los elementos <a> estén señalados por el ratón cumplirán las instrucciones
