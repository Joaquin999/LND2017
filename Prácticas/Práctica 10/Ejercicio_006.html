La instrucción a:visited significa que todos los elementos <a> que hayan sido visitados cumpliran las instrucciones que se dispongan, en este caso se dispone que no tendrán ningún tipo de adornos de texto y su texto será de color azul

La instrucción a:link significa que todos los elementos <a> con un enlace externo cumpliran las instrucciones que se dispongan, en este caso se dispone no tendrán ningún tipo de adornos de texto y su texto será de color rojo

La instrucción a:hover significa que todos los elementos <a> que se encuentren señalados por el ratón cumpliran las instrucciones que se dispongan, en este caso se dispone que tendrán un subrayado como adorno de texto y su texto será de color rojo
